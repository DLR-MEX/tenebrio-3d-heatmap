Objetivo:
Mantener y mejorar el sistema de visualización 3D de temperatura
para un cuarto de cría de tenebrios.

El sistema obtiene datos desde Ubidots usando MQTT
y renderiza un mapa de calor volumétrico 3D con Plotly.js.

Nunca cambiar la arquitectura sin instrucción explícita.
